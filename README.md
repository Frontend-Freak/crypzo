# Crypzo

Приложение для отслеживания криптовалютных активов. Позволяет искать монеты, добавлять их в избранное и смотреть рыночные данные. 

![demo](./assets/demo.png)

## Возможности:
- Поиск монет через CoinGeco API
- Авторизация через Supabase
- Добавление монет в избранное

## Технологии:
- TypeScript
- React
- Redux Toolkit
- Supabase
- Material UI

## Установка и запуск:
git clone https://github.com/Frontend-Freak/crypzo.git
cd crypzo
npm install
npm run dev

## Тесты:
npm run test
